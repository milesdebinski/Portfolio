About

SpendAbroad is an app that can be used to check current exchange rates, as well as see which countries the currency can be used in.

The app is supported by 2 API's: fixer.io and rest countries. The input is an amount of money and choosen currencies to exchange from and to, and the output is amount of exchanged money and list of countries wher you can use certain currency.
